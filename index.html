CSS Selectors Cheat Sheet

Selector Example Description
.class .button {color: red; } Targets elements with the specified class.
#id #header {font-size: 20px; } Targets elements with a specific ID.
element p { margin: 10px; } Targets all instances of the specified element.
child ul > li { list-style: none; } Targets direct child elements.
descendant div p { color: blue; } Targets all matching elements inside a specified element.
pseudo-class a:hover { color: green; } Applies styling to an element in a specific state.
attribute input[type="text"] { border: 1px solid balck; } Targets elements with a specific attribute.
